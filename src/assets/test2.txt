
# Технические заметки
=====================================

## Структура проекта
- Frontend: React 18 + TypeScript
- Стилизация: Tailwind CSS
- Компоненты: shadcn/ui
- Анимации: CSS transitions + tailwindcss-animate

## API эндпоинты:
- GET /api/prizes - получение списка призов
- POST /api/spin - запись результата вращения
- GET /api/history - получение истории вращений

## Производительность:
- Оптимизация рендеринга с useCallback/useMemo
- Ленивая загрузка больших компонентов
- Кэширование результатов с React Query
