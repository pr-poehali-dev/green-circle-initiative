
# ВИДЫ КОКТЕЙЛЬНЫХ БОКАЛОВ

## ХАЙБОЛ (HIGHBALL)
Высокий узкий стакан для лонгдринков. Идеален для коктейлей с содовой или газировкой.
Объем: 240-350 мл
Для коктейлей: Мохито, Джин-тоник, Куба Либре

## РОКС (OLD FASHIONED)
Низкий широкий стакан для крепких напитков со льдом.
Объем: 180-300 мл
Для коктейлей: Виски on the rocks, Негрони, Олд Фэшнд

## КОКТЕЙЛЬНЫЙ (МАРТИНИ)
V-образный бокал на ножке для коктейлей без льда.
Объем: 90-150 мл
Для коктейлей: Мартини, Дайкири, Манхэттен

## МАРГАРИТА
Широкий бокал на ножке с двумя чашами, верхняя шире нижней.
Объем: 250-350 мл
Для коктейлей: Маргарита, Замороженные дайкири

## ХУРРИКЕЙН (HURRICANE)
Высокий изогнутый бокал на короткой ножке.
Объем: 400-450 мл
Для коктейлей: Пина Колада, Май Тай, тропические коктейли
