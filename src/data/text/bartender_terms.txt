
# СЛОВАРЬ БАРМЕНСКИХ ТЕРМИНОВ

- **Шейк** - энергичное взбалтывание напитка в шейкере со льдом
- **Стир** - медленное перемешивание напитка барной ложкой
- **Билд** - послойное построение коктейля прямо в бокале
- **Мадл** - разминание фруктов или трав для извлечения ароматов
- **Флоат** - наслаивание компонента поверх напитка
- **Римминг** - обработка края бокала солью, сахаром или специями
- **Твист** - цедра цитрусовых, скрученная для выделения ароматических масел
- **Гарниш** - украшение коктейля (фрукты, травы, специи)
- **Джиггер** - мерный инструмент для точного отмеривания жидкостей
- **Стрейн** - процеживание коктейля для удаления льда или частиц фруктов
- **Дэш** - небольшое количество жидкости (несколько капель)
- **Драй шейк** - взбалтывание в шейкере без льда
- **Роллинг** - перекатывание напитка между стаканами для перемешивания
- **Спритц** - легкое опрыскивание поверхности коктейля ароматическими веществами
