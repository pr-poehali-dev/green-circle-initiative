
# СОВЕТЫ ДЛЯ НАЧИНАЮЩИХ БАРМЕНОВЫЫЫ

## ОСНОВНЫЕ ПРАВИЛА
- Всегда используйте свежие ингредиенты для наилучшего вкуса
- Измеряйте компоненты с помощью джиггера для точности
- Пробуйте каждый коктейль перед подачей (через трубочку)
- Поддерживайте чистоту рабочего места
- Правильно выбирайте бокалы для конкретных коктейлей

## МИНИМАЛЬНЫЙ НАБОР ИНСТРУМЕНТОВ
- Шейкер (бостонский или кобблер)
- Джиггер для измерения
- Барная ложка
- Стрейнер (ситечко)
- Мадлер для разминания
- Нож и разделочная доска
- Набор базовых бокалов

## ТЕХНИКИ СМЕШИВАНИЯ
- **Шейк** - для коктейлей с соками, сиропами, яйцами
- **Стир** - для прозрачных, алкогольных коктейлей
- **Билд** - для простых напитков, построенных в бокале
- **Мадл** - для извлечения эфирных масел из трав/фруктов
- **Блендер** - для замороженных напитков

## РАСПРОСТРАНЕННЫЕ ОШИБКИ
- Перенасыщение льдом
- Неправильные пропорции
- Избыточное/недостаточное перемешивание
- Смешивание несовместимых ингредиентов
- Использование старых/испорченных компонентов

## ПРОФЕССИОНАЛЬНЫЕ СЕКРЕТЫ
- Предварительно охлаждайте бокалы
- Используйте крупный лед для медленного таяния
- Освойте технику наслаивания ликеров
- Изучите базовые пропорции классических коктейлей
- Экспериментируйте с ароматическими биттерами
