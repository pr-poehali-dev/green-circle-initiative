# СОВЕТЫ ПО ГАРНИРОВКЕ КОКТЕЙЛЕЙ

## ОСНОВНЫЕ ПРИНЦИПЫ
- Гарнир должен дополнять вкус напитка, а не только украшать его
- Используйте свежие и качественные ингредиенты
- Соблюдайте пропорции: гарнир не должен мешать пить коктейль
- Учитывайте цветовой контраст для визуальной привлекательности
- Подготавливайте гарниры непосредственно перед подачей

## ПОПУЛЯРНЫЕ ГАРНИРЫ И ИХ ИСПОЛЬЗОВАНИЕ
- **Цитрусовая цедра**: выжмите эфирные масла над поверхностью напитка перед добавлением
- **Оливки и лук**: используйте для савори-коктейлей типа Мартини или Гибсона
- **Ягоды**: должны быть свежими и плотными, идеальны для фруктовых коктейлей
- **Травы (мята, розмарин, тимьян)**: слегка помните перед использованием для усиления аромата
- **Солевой/сахарный ободок**: смочите край бокала лимоном и окуните в соль или сахар
- **Коктейльные вишни**: используйте качественные, без искусственных красителей
- **Цветы**: применяйте только съедобные, специально выращенные для кулинарии

## ТЕХНИКИ ГАРНИРОВКИ
- **Подвешивание**: фиксация гарнира на краю бокала
- **Плавающая гарнировка**: размещение легких элементов на поверхности напитка
- **Погружение**: полное погружение гарнира в напиток для насыщения вкуса
- **Нанизывание**: использование шпажек для сложных композиций
- **Ободки**: декорирование краев бокала солью, сахаром или специями

## ИНСТРУМЕНТЫ
- Канеллер для создания цедры
- Острый нож для чистых срезов
- Пинцет для деликатной работы
- Коктейльные шпажки и пики
- Маленькие формочки для льда и желе
