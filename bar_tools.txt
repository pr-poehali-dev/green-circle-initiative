# Базовые барные инструменты

- Шейкер (бостонский или кобблер)
- Джиггер для точного измерения
- Барная ложка для перемешивания
- Мадлер для разминания фруктов и трав
- Стрейнер для фильтрации
- Нож и разделочная доска
- Пресс для цитрусовых
- Щипцы для льда
- Мерные стаканы
- Контейнеры для гарниров
