# ВИДЫ КОКТЕЙЛЬНЫХ БОКАЛОВ

## Классические бокалы
- **Мартини** — V-образный бокал на тонкой ножке для коктейлей без льда
- **Хайбол** — высокий цилиндрический стакан для лонг-дринков
- **Олд фэшн** — низкий толстостенный стакан для крепких напитков
- **Коктейльная рюмка** — маленький бокал на ножке для подачи дижестивов

## Специализированные бокалы
- **Маргарита** — широкий бокал с углублением для солевой каймы
- **Харрикейн** — изогнутый высокий бокал для тропических коктейлей
- **Коллинз** — высокий узкий стакан для освежающих напитков
- **Купе** — широкая неглубокая чаша на ножке для шампанских коктейлей

## Выбор бокала
Правильный бокал не только красиво подает напиток, но и влияет на его вкусовые качества. Форма бокала определяет:
- Температуру напитка
- Концентрацию ароматов
- Скорость таяния льда
- Удобство употребления
